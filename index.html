Tentang Saya

Mengenal Firman lebih dekat

Passion dalam Teknologi & Inovasi

Firman merupakan seorang lulusan Teknik Elektro Universitas Negeri Yogyakarta dengan ketertarikan pada pengembangan sistem kelistrikan, sistem kendali, Internet of Things, dan machine learning. Selama masa studi, Firman terbiasa mengerjakan berbagai proyek berbasis mikrokontroler dan sistem cerdas yang mengintegrasikan perangkat keras dan perangkat lunak untuk menyelesaikan permasalahan teknis secara terstruktur dan efisien.

Firman memiliki pengalaman magang di PT Industri Kereta Api (Persero) pada bagian instalasi sistem kelistrikan. Pengalaman ini membekali Firman dengan pemahaman mengenai penerapan teknik elektro di lingkungan industri, termasuk pentingnya keandalan sistem dan standar keselamatan kerja. Selain itu, Firman terbiasa menggunakan PLC, mikrokontroler, serta berbagai perangkat lunak teknik untuk perancangan, simulasi, dan implementasi sistem.

Ketertarikan dan pengalaman tersebut tercermin dalam proyek skripsi berjudul "Pengembangan Prototipe Sistem Pemilahan Sampah Otomatis Menggunakan Metode Convolutional Neural Network pada Raspberry Pi 3 Model B dan NodeMCU ESP8266". Melalui proyek ini, Firman mengintegrasikan sistem embedded, Raspberry Pi, ESP8266, dan machine learning dalam satu kesatuan sistem cerdas.
